# 网站名字建议

> 核心诉求：传达「我们是 Miniscript 的布道者/向导，而非发明者」，用拼接词做单词品牌名。

---

## 🔥 围绕 "spend"（花费条件是网站灵魂）

| 名字 | 拼接逻辑 | 含义 |
|------|----------|------|
| **Spendwise** | spend + wise | 「懂花费」，直击核心：谁能花、何时花、怎么花 |
| **Spendora** | spend + explora / pandora | 打开花费条件的潘多拉之盒 |
| **Spendcraft** | spend + craft | 花费条件的手艺/技艺 |
| **Spendpath** | spend + path | 花费路径，与右栏 Spending Paths 呼应 |

## 🔍 围绕 "script"（技术关键词）

| 名字 | 拼接逻辑 | 含义 |
|------|----------|------|
| **Scriptwise** | script + wise | 智慧的脚本 |
| **Scriptorium** | 拉丁语原词 | 历史上专门研读手稿的地方，天然学术气质 |
| **Scriptcraft** | script + craft | 脚本匠艺 |
| **Scriptory** | script + story / observatory | 脚本的故事或观测站 |
| **Scriptual** | script + textual / ritual | 像「经文的」，庄重又有趣味 |

## 🔬 围绕 "mini"（保留 Miniscript 辨识度）

| 名字 | 拼接逻辑 | 含义 |
|------|----------|------|
| **Miniscope** | mini(script) + scope | 窥探 Miniscript 的镜头 |
| **Miniwise** | mini(script) + wise | 理解 Miniscript 的智慧 |
| **Minigrok** | mini + grok | grok = 极客黑话「深刻理解」 |

## 🔑 围绕 Bitcoin 语义

| 名字 | 拼接逻辑 | 含义 |
|------|----------|------|
| **Satscript** | satoshi + script | 比特币味道拉满 |
| **Lockcraft** | lock + craft | 比特币本质是锁与钥匙 |
| **Keypath** | key + path | 密钥路径，也是 Taproot 术语双关 |

## 🎲 更大胆的方向

| 名字 | 拼接逻辑 | 含义 |
|------|----------|------|
| **Unscript** | un + script | 把晦涩的 Script 变得可理解 |
| **Inscript** | in + script | 深入脚本，也像 insight |
| **Grokscript** | grok + script | 极客圈一看就懂 |

---

## ⭐ Top 3 推荐

1. **Spendwise** — 最贴网站灵魂，朗朗上口，非技术用户也秒懂
2. **Miniscope** — 保留 Mini 辨识度，「用镜头看清 Miniscript」
3. **Scriptorium** — 最有气质，学术感强，适合科普品牌
